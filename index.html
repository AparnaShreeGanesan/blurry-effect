Code samples

The examples below show what an integration might look like if this were exposed as a hosted API. Replace placeholders with your own credentials and endpoint. 

import requests

endpoint = "https://api.example.com/v1/blurry-image-identifier/invoke"
token = "YOUR_BEARER_TOKEN"

payload = {
    "data": "https://example.com/your-image.jpg"
}

response = requests.post(
    endpoint,
    json=payload,
    headers={"Authorization": f"Bearer {token}"}
)

print(response.json())
fetch("https://api.example.com/v1/blurry-image-identifier/invoke", {
    method: "POST",
    headers: {
        "Authorization": "Bearer " + "YOUR_BEARER_TOKEN",
        "Content-Type": "application/json"
    },
    body: JSON.stringify({
        data: "https://example.com/your-image.jpg"
    })
})
    .then(response => response.json())
    .then(console.log);
curl -X POST \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_BEARER_TOKEN" \
  -d '{"data": "https://example.com/your-image.jpg"}' \
  https://api.example.com/v1/blurry-image-identifier/invoke

Why detect blurry images

Better user uploads

Catch low-quality images as they are uploaded so users can submit sharper photos before they are stored or processed.

Higher quality datasets

Filter out blurry samples from training datasets to improve downstream model performance on vision tasks.

Automated QA checks

Integrate blur detection into pipelines for ecommerce, ID verification, and photography workflows.
